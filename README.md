# dotfiles

A simple script to automate your macOS development environment setup.

## 🚀 Usage

Open your terminal and run the following command:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/laziness-works/dotfiles/refs/heads/main/install.sh)"
```

## 📦 What's Included?

- Homebrew (with automatic PATH configuration)
- Rosetta 2
